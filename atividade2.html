código de programação é um conjunto de instruções escritas em uma linguagem específica que permite que um computador execute tarefas. É como uma linguagem que os computadores entendem, permitindo que os programadores criem softwares, aplicativos e sistemas.

linguagens de programação são sistemas formais, como um idioma artificial, que usam palavras-chave, símbolos e regras para dar instruções a um computador, permitindo a criação de softwares, aplicativos e websites.

linguagem JavaScript (ou JS) é uma tecnologia principal da web que adiciona interatividade e dinamismo a páginas e aplicações, desde animações e mapas interativos até menus suspensos e formulários.

Python é uma linguagem de programação de alto nível, interpretada, de código aberto e multiparadigma, criada por Guido van Rossum no final dos anos 80.

TypeScript é uma linguagem de programação de código aberto, criada pela Microsoft, que é um superconjunto do JavaScript.
